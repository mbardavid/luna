# M1 Pre-Publish Checklist

Status: Draft operator checklist  
Reference: `docs/luna-x-growth-charter.md`  
Mode: use for review only until freeze is explicitly lifted

## Fast gate
If any answer below is **No** or **Unclear**, hold the item for review.

## 1) Pillar fit
- [ ] The draft clearly maps to at least one approved pillar/content theme.
- [ ] The draft does not drift into an unapproved topic.
- [ ] The audience fit is identifiable and consistent with the distribution map.

## 2) Evidence quality
- [ ] The core point is grounded in a real observation, example, log, data point, or explicitly stated uncertainty.
- [ ] Any factual claim can be supported if asked.
- [ ] The post does not imply evidence that is missing.

## 3) Tone check
- [ ] Tone is concise, technical, candid, and non-manipulative.
- [ ] The draft avoids clickbait, engagement bait, or vanity-metric framing.
- [ ] The draft does not provoke conflict for reach.

## 4) Claim safety
- [ ] No guarantee, promise, or overconfident claim is being made.
- [ ] No financial advice, return promise, or unverifiable performance flex is present.
- [ ] Any market-related statement includes clear uncertainty and stays evidence-backed.

## 5) Charter constraints
- [ ] The draft respects the freeze charter in `docs/luna-x-growth-charter.md`.
- [ ] The item is not a growth hack, outbound acquisition tactic, or public positioning shift.
- [ ] The draft does not create identity/persona mismatch risk.

## 6) Approval state
- [ ] Current state is recorded: draft / under review / approved / held / paused.
- [ ] Required human approval has been obtained for any public action.
- [ ] If approval is missing, the item remains unpublished.

## 7) Measurement plan
- [ ] A measurement note exists for what success/failure would look like.
- [ ] The intended audience and expected signal are documented.
- [ ] Post-level tracking fields are ready in `m1-metrics-tracker.md`.

## 8) Final operator decision
- [ ] Publish
- [ ] Hold for edits
- [ ] Reject
- [ ] Escalate for human review
